Cmd中mvn_compile出现的问题

在cmd上运行mvn compile时出现编译错误

No compiler is provided in this environment. Perhaps you are running on a JRE rather than a JDK?

输入mvn -v查看

查看环境变量

将JAVA_HOME路径修改为jdk的

重启cmd输入java -version检查是否配置成功

mvn -v

再次编译,成功
